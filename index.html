#我是如何搭建这个博客

##1. 来由
为什么要写博客呢？从我的角度来看就是为了更好的把自己脑子中想的内容用文字的形式表达出来，也是为了日后与他人更好的交流沟通想法。平时，我自己查询内容的时候更倾向于别人的博客，因为在博客中，博主会用通俗易懂的语言描述加上自己的想法。这很容易想到TED的主旨

Ideas worth spreading

##2. 经过
搭建博客有很多种方式，我采用的是GitHub Pages + hexo 途中没有任何多余的花费，除了在腾讯云购买的1RMB/年的域名（备案什么的太麻烦，还在路上）

搭建过程中主要参考了这篇博文：可能是最详细的 Hexo + GitHub Pages 搭建个人博客的教程

一路搭建的还是算蛮顺利的，花费时间最长应该还算是主题的选取吧，想到日后肯定还是会更改主题的，个人比较倾向于简单，所以选择了这款主题。

其中搭建的一些指令第一次接触看着会很懵逼，但是对于博客的搭建没有影响，但还是需要学习

  1. Markdown的语法知识（简书介绍的就够用了）

  2. 简单的Git操作。
